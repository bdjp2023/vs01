cd Desktop/スクレイピングツール
pip install -r requirements.txt


使い方① 手動で1回だけ実行する方法（おすすめ初心者）
コマンドプロンプトを開く
以下の命令を入力：

cd /home/bdjp/Desktop/vs01/app/sp00
python main.py


使い方② 見張り番（watcher）を使う方法（自動で動く）
この方法を使うと、input.jsonを保存した瞬間に自動でスクレイピングが始まります。

起動方法
コマンドプロンプトを開く
以下の命令を入力：

cd home/bdjp/Desktop/vs01/app/sp00
python watcher.py

「見張り番を起動しました。input.jsonを待機しています...」と表示されたら成功です。
これで黒い画面は開いたままにしておきます。

自動で動かす手順
input.jsonをメモ帳やVisual Studio Codeで開く
中身を編集して保存する
数秒後に「input.jsonが更新されました！スクレイピングを開始します」と表示されて自動で動き出します


結果の見方
処理が終わると以下のファイルができます：

output_2026-06-06.csv ← これが取得したデータ（Excelで開けます）
scrape.log ← 何が起きたかの記録（メモ帳で開けます）


止める方法
見張り番（watcher.py）を止めたいとき
黒い画面をクリックして Ctrl + C を押してください。

何度も「Ctrl + C」を押して「^C」と表示されたら停止しています。



よくあるトラブルと解決法
Q. 「pythonが見つかりません」と言われた
→ Pythonをインストールし直して、「Add python.exe to PATH」にチェックを入れてください。

Q. エラーが出て止まる
→ 黒い画面のエラーメッセージを全部コピーして、AI（Claude）に「このエラーを直したバージョンをください」とお願いしてください。

Q. ブラウザがすぐに閉じてしまう
→ main.pyを開いて、headless=Falseになっているか確認してください。

Q. watcherが反応しない
→ watcher.pyを一旦Ctrl+Cで止めて、もう一度 python watcher.py で起動し直してください。
